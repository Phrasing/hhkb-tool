## KEY MAPPING WHEN RSHIFT IS HELD
| Original Key | ⇧ | Remapped Key |
|--------------|---|--------------|
| W            | ⇧ | ↑            |
| A            | ⇧ | ←            |
| D            | ⇧ | →            |
| S            | ⇧ | ↓            |
| E            | ⇧ | End          |
| Q            | ⇧ | Home         |
| R            | ⇧ | Page Up      |
| F            | ⇧ | Page Down    |
| Z            | ⇧ | Delete       |
